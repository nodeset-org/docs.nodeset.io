# Security Information

### Audits

Constellation was audited for three months by three teams of leading independent security professionals. To read the full reports, please see [the Constellation repository's `audits` directory](https://github.com/nodeset-org/constellation/tree/main/audits).

### Bug Bounties

{% hint style="info" %}
Constellation is now deprecated. While the contracts remain deployed forever on Ethereum, the bug bounty program has ended and no reports will be considered. Please **do not contact NodeSet with bug reports related to Constellation** as they will be ignore&#x64;**.**
